Event Delegation

Event Delegation is a JavaScript technique where a single event listener is attached to a parent element instead of attaching separate event listeners to each child element.

It works because of event bubbling, where an event triggered on a child element automatically bubbles up to its parent.

Advantages

  • Improves performance by using fewer event listeners.
  • Works for dynamically created elements.
  • Makes the code cleaner and easier to maintain.

Syntax

parent.addEventListener("click", function(event){

    if(event.target.matches(".child")){
        // Your code here
    }

});

Example

<ul id="list">
    <li>Apple</li>
    <li>Mango</li>
    <li>Banana</li>
</ul>

const list = document.getElementById("list");

list.addEventListener("click", function(event){

    console.log(event.target.textContent);

});

Output: Clicking on any list item prints its text (Apple, Mango, or Banana) to the console using a single event listener attached to the parent <ul>.

# 💡 Bulb On/Off Project

A simple interactive Bulb On/Off project built using HTML, CSS, and JavaScript. This project demonstrates DOM manipulation, event handling, and dynamic styling in JavaScript.

## 🚀 Features

* Turn the bulb ON and OFF using a button.
* Interactive user interface.
* DOM element selection and manipulation.
* Event handling with JavaScript.
* Beginner-friendly project for learning web development fundamentals.

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

├── index.html
├── style.css
├── script.js
└── README.md

## 🎯 Learning Outcomes

Through this project, I learned:

* How to select DOM elements using JavaScript.
* How to use event listeners.
* How to modify CSS properties dynamically.
* How to create interactive web applications.
* Basic project structuring for web development.

## 🔧 Future Improvements

* Support keyboard controls.
* Add brightness adjustment slider.
* Create multiple bulb controls.

## 📸 Preview

A simple bulb interface with a button that allows users to toggle the bulb state between ON and OFF.

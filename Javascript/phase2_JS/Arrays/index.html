Array
An array is a collection of ordered set of elements.It can contain values of different datatypes.
    Mutating methods of array
  1. push(x): adds at end
  2. pop(): removes from end and returns it.
  3. unshift(): adds at beginning
  4. shift(): deletes 1st element of an array
  5. sort(): arrange in ascending order (Treats numbers as strings)
  6. reverse(): reverses the order of array
  7. splice(): to add or remove element(s) at/from desired index.
    • To add: splice(startIndex, deleteCount, ...items)
    • To delete: splice(startIndex, deleteCount)
    Non - Mutating methods of array
  1. slice(startIndex,endIndex): returns new array
  2. concat(): add two arrays
  3. includes(): returns boolean value, checks if element is present or not
  4. indexOf(): returns index of element if element is present else -1
  5. join(): joins the elemets of an array and returns string

Functions in JavaScript

Functions are reusable blocks of code that perform a specific task.

    They are:
  • reusable
  • accessible
  • Systematic

Function Declaration

        function functionName(parameters) {
            // code to be executed
        } 
        functionName(arguments);  // calling the function

Function Expression/ First Class Functions

The function is assigned to a variable.

        const functionName = function(parameters) {
            // code to be executed
        };
        functionName(arguments);  // calling the function

Arrow Function

A shorter syntax for writing functions.

        const functionName = (parameters) => {
            // code to be executed
        };
        functionName(arguments);  // calling the function

IIFE: Immediately Invoked Function Expression

A function that is executed immediately after it is defined.

        (function functionName(parameters) {
            // code to be executed
        })(arguments);  // calling the function

Callback Function

Callback function are the functions that are passed as argument to another function.

 
        function callback(name, callback){
            callback(name); //main("Hello")
        }
        function main(a){
            console.log(a);
        }
        callback("Hello", main); 
        //main i.e. function name is passed as argument that calls the main() function inside callback() function.

Output:"Hello"

Higher-Order Function

A function that takes another function as an argument or returns a function as a result.

        function higherOrderFunction(callback) {
            // code to be executed
            callback();
        }

        higherOrderFunction(function() {
            // code to be executed as a callback
        });
